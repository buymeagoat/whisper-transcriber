Welcome to the Whisper Transcriber API - a modern, mobile-first audio transcription service.

Quick Start

GET /docs - Interactive API documentation (Swagger UI)
GET /redoc - Alternative API documentation (ReDoc)
GET /health - Service health check

Authentication

POST /auth/login - Get access token
GET /auth/me - Get current user info

Core Features

POST /jobs/upload - Upload audio for transcription
GET /jobs/{job_id} - Get transcription job status
GET /jobs/{job_id}/download - Download transcription results

System Information

Version: 1.0.0

Backend: FastAPI + SQLAlchemy + Celery

AI Model: OpenAI Whisper

Database: SQLite (Development)

Visit /docs for complete API documentation
